What is Blender?

Blender is a 3D computer graphics and production software aimed at creating and animating 3D games and strategic images. It offers us complete 3D modeling development features, from hundreds to thousands of features (image and video production, game modeling, textualization, painting, etc.) and completely free, making it free software.

Types of modeling in Blender:

A 3D model is a mathematical representation of three-dimensional surfaces. This representation can be created manually or automatically. These are the three most common ways to model these days.:

Polygonal modeling:

Points/vertices, connected to each other through lines/edges, delimit faces forming a polygonal mesh. Currently, the vast majority of 3D models are built using textured polygonal models. However, polygons are flat, they can only represent curved surfaces using many polygons.

Curve modeling:

Surfaces are defined by curves that are manipulated through control points, the curve follows the points. There are different types of curves, including NURBS.

Digital sculpting:

Relatively recent method. There are currently three subtypes: Displacement, Volumetric, Dynamic tesselation.

Some Blender tools:
Extrusion:

Extrusion is an essential modeling tool. Allows you to create a new mesh from the existing mesh. Projects faces, edges, or vertices the same size and shape as the selected ones. For example, extrusion onto a square face would create a cube, etc.

Bevels ou Chamfers:

Beveling/Beveling (beveling or chamfering) allows you to cut the edges of an object. You can use the Bevel tool only on vertices (instead of edges), with different angle limits, different cuts, widths, etc.

Loops (Edge e Cuts):

Edge Loop can be defined as a set of edges connected through a surface. Usually, the last edge is connected to the first, creating a loop. The edges of a model must follow in the form of loops so that a character's animation results without the errors and deformations that result from the intertwined mesh. Loop Cuts is a modeling tool that allows you to cut a loop of faces by inserting a new Edge Loop.

Non-Destructive Modeling e Modifiers:

Modifiers are automatic operations that affect an object in a non-destructive, non-definitive way. Modifiers allow you to perform various operations without affecting the object's base topology, its real geometry. If desired, Modifiers can be made definitive, applied (Apply).

Textualization in Blender:

When adding a text object in Blender (SHIFT+A) an object with the text “Text” is automatically inserted. To edit text, write new words or delete, switch to Edit Mode and use the keyboard as normal. In Object Mode, text works very similarly to other objects, you can move, rotate, resize, insert keyframes, apply materials.
